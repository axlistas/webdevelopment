Opdracht 1:

De blokjes zijn de html elementen die zichtbaar zijn in het html document. De lijnen geven aan dat binnen in deze html elementen er zich nog andere elementen kunnen bevinden, dit noemen we nesten.

Opdracht 2:

We zien dat bij het verwijderen van de elementen uit de DOM-tree gaan we zien dat er elementen uit de statische pagina gaan verdijnen.

Opdracht 3:

Ik zie dat de accordeon bestaat uit een grote div, deze is opgedeeld in andere H3's (headings) en div's, bij het aanklikken van een andere H3 van de accordeon gaat de DOM-tree zich aanpassen en toont hij de div die hierbij hoort.

Ik zie ook dat "aria-expanded=" een true of false variabele heeft, deze geeft aan of het element actief en zichtbaar moet zijn of niet en vinden we terug in het H3 element.

Als ik kijk naar de source code van het document dan zien we dat dit er veel eenvoudiger uitziet dan wanneer we het via Chrome Developer Tools bekijken.

Dit omdat we gebuik maken van een externe library "jquery" die we importeren aan de hand van een script tag in de "head" van onze html pagina.

Als we dan in onze div een parameter id="accordion" en "expanded" gaan gebruiken dan gaat onze site aan de hand van Jquery de juiste structuur kunnen inladen.
