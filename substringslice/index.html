Verschil tussen JavaScript substring en slice

JavaScript substring Methode

De substring methode retourneert een deelstring tussen de twee opgegeven indexen, of tot het einde van de string. De eerste index is de startpositie en de tweede index (optioneel) is de eindpositie (niet inbegrepen).

Syntax: str.substring(indexStart, indexEnd)

Voorbeeld:


const str = "JavaScript";
const result = str.substring(0, 4);
console.log(result); // "Java"

Als indexEnd wordt weggelaten, wordt de rest van de string vanaf indexStart tot het einde van de string genomen.

JavaScript slice Methode

De slice methode retourneert een deelstring van de oorspronkelijke string. Het accepteert twee parameters: de startindex en de eindindex (optioneel). Net als bij substring is de eindindex niet inbegrepen.

Syntax: str.slice(beginIndex, endIndex)

Voorbeeld:


const str = "JavaScript";
const result = str.slice(0, 4);
console.log(result); // "Java"

Als endIndex wordt weggelaten, wordt de rest van de string vanaf beginIndex tot het einde van de string genomen.

Belangrijkste Verschillen

Hoewel substring en slice op het eerste gezicht vergelijkbaar lijken, zijn er enkele belangrijke verschillen:

  • slice kan negatieve indexen accepteren, wat betekent dat je kunt tellen vanaf het einde van de string. substring accepteert geen negatieve indexen en behandelt negatieve waarden als 0.
  • Als de startindex groter is dan de eindindex, wisselt substring de indexen om. slice doet dit niet en retourneert een lege string.

Voorbeeld met negatieve indexen:


const str = "JavaScript";
const resultSlice = str.slice(-6, -1);
console.log(resultSlice); // "Scrip"

const resultSubstring = str.substring(-6, -1);
console.log(resultSubstring); // "JavaScript"

Zoals je kunt zien, behandelt slice negatieve indexen door vanaf het einde van de string te tellen, terwijl substring negatieve indexen als 0 behandelt.
