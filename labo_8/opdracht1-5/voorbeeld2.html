Waarom staat de tekst in de blokken horizontaal gecentreerd?

  • Dit komt omdat we in de css het element "text-allign" op center hebben staan.

Waarom heeft property vertical-align:center geen effect?

Dit werkt niet omdat we geen property width hebben die hier op inspeelt.

Wat doet vertical allign middle?

vertical allign middle zorgt er voor dat de middenlijn van alle blokken doorgetrokken wordt, we zien dat de blokken mooi naast elkaar staan en dat er telkens even veel spatie onder de midden lijn als boven is voor elke blok.

Hoe centreren we dan wel de tekst verticaal in een element?

Als we de line-height van de tekst gelijk stellen aan de height van het element zelf dan positioneert de tekst zich mooi in het midden.

Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren?

Vaak maken ze gebruik van het css element display: table-cell. Deze techniek werkt goed voor het centreren van meerdere regels tekst omdat de tabel-celbenadering automatisch de hoogte van de container aanpast aan de inhoud, waardoor het verticaal centreren van de inhoud wordt vergemakkelijkt.

Waarom krijgt de regel voor class 'tall' voorrang?

De class tall komt later voor in het css document, deze wordt dus als laatste geinisialiseerd van beiden en gaat dus de andere overschrijven.
