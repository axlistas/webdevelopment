3.4.4 Semantische elementen

<section>

Dit gebruiken we om gerekateerde stukken informatie te groeperen, en is te vergelijken met een div

Voorbeeld: Een reeks nieuwsberichten, reacties op een bericht

Alleen gebruikbaar als er een header(h1,..) element aanwezig is

<article>

Hierin plaatsen we een stuk informatie dat onafhankelijk is van de rest, we zouden het kunnen plaatsen op een andere site zonder context

<aside>

Vaak gebruiken we dit om achtergrond informatie ge geven over een bepaald onderwerp, het kan perfect apart bekeken worden los van de rest vd pagina

Hier plaatsen we vaak sidebars of marges

<hgroup>

Een groepering van header elementen die bij elkaar horen

<header>

Bevat vaak een baner, logo's en een zoekformuleer.

Maar we kunnen dit ook gebruiken om een hoofding binnen in een artikel te definieeren.

Dit kan bevatten: figure tag; h1,h2 ect elementen; nav element; p element

<footer>

Dit is op veel pagina's onderaan te vinden, contact info, copyright rechten etc..

<nav>

Dit gebruiken we om een navigatie menu weer te geven, belangrijke links naar een andere pagina / site

<time>

Dit gebruiken we om een datum of tijd weer te geven

3.7 Hyperlinks en anchors

We maken gebruik van een <a> tag om links te leggen naar andere adressen (mail/web) of een inline page adres (anchor)

Om naar een andere link te gaan gebruiken we:

<a href"url">tekst"</a>

Om naar een mail te sturen gebruiken we:

<a href"mailto:mail">Verstuur een email"</a>

Om te scrollen naar een bepaald punt op de pagina:

<a href"#id">Scrol naar ..."</a>
